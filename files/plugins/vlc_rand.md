## vlc_rand_bytes {#vlc_rand_bytes}

```c
VLC_API void vlc_rand_bytes(void *buf, size_t len);
```

### 函数描述
该函数用于生成随机字节序列，并将生成的随机字节填充到指定的缓冲区中。

### 函数参数说明

| 参数名 | 类型   | 描述                   |
| ------ | ------ | ---------------------- |
| buf    | void * | 指向目标缓冲区的指针，用于存储生成的随机字节。 |
| len    | size_t | 要生成的随机字节的长度。 |

### 函数返回值
该函数没有返回值。
