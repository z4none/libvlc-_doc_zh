`abort` 函数用于立即终止程序的执行。它不会正常地终止程序，而是通过发送一个未处理的信号（通常是 `SIGABRT`）来强制终止程序。调用 `abort` 函数通常用于在程序遇到无法恢复的错误时强制退出。

### 函数参数说明
| 参数名 | 类型 | 描述 |
|--------|------|------|
| 无     | 无   | 无   |

### 函数返回值
`abort` 函数不会返回，因为它会导致程序的终止。调用 `abort` 后，程序将立即终止，不会返回到调用点。
