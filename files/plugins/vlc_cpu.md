## vlc_CPU {#vlc_CPU}

```c
VLC_API unsigned vlc_CPU(void);
```

### 函数描述
该函数用于检测当前系统的 CPU 特性。它返回一个无符号整数，表示系统支持的 CPU 特性。

### 函数参数说明

| 参数名 | 类型 | 描述 |
|--------|------|------|
| void   | void | 无参数 |

### 函数返回值
该函数返回一个无符号整数，表示系统支持的 CPU 特性。返回值的具体含义如下：

- **返回值为 0**：表示系统不支持任何特殊的 CPU 特性。
- **返回值为非 0**：表示系统支持特定的 CPU 特性，具体特性由返回值的位掩码表示。
