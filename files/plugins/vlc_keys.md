## hotkey 结构定义 {#hotkey}

```c
struct hotkey
{
    const char *psz_action;
};
```

### 描述
`hotkey` 结构体用于定义一个热键（快捷键）。该结构体包含一个指向字符串的指针，该字符串表示与热键关联的动作或命令。

### 成员说明

| 成员变量    | 类型            | 描述                                                                 |
|-------------|-----------------|----------------------------------------------------------------------|
| `psz_action`| `const char *`  | 指向一个字符串的指针，该字符串表示与热键关联的动作或命令。            |

### 返回值
该结构体本身不返回值，它用于存储热键相关的信息。
